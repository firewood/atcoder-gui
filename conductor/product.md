# Initial Concept

A browser automation tool for AtCoder with CLI interface built with Node.js, TypeScript, and Playwright.

## 1. Product Vision
AtCoderでの競技プログラミング体験を劇的に効率化し、ブラウザ操作の手間を省いてコーディングに集中できる環境を提供します。

## 2. Target Users
- **初心者競技プログラマー:** 環境構築や問題文の確認、テスト、提出といった一連の流れを簡単にしたい人。
- **ヘビー競技プログラマー:** コンテスト中の操作を極限まで自動化し、数秒の短縮を目指す人。

## 3. Core Goals
- **コンテストサイクルの高速化:** 問題の取得、サンプルのテスト、最終的な提出までを一貫して自動化します。
- **ブラウザ操作の抽象化:** Playwrightを活用し、手動でのブラウザ操作をCLIからシームレスに行えるようにします。
- **セッション管理の自動化:** ログイン状態を永続化し、シームレスなセッション復元を実現します。

## 4. Key Features
- **問題解析 (Analyzer):** 問題文からサンプルケースや入出力形式を自動的に抽出します。
- **コード生成 (Generator):** 抽出したデータに基づき、C++やPythonなどのテンプレートからコードを自動生成します。
- **CLI インターフェース:** 対話形式のCLIにより、素早くコマンドを実行可能です。
- **ブラウザ自動化:** Playwrightを利用した安定したブラウザ制御。

## 5. Future Scope
現在はAtCoderに特化した最高の支援ツールを目指し、将来的な拡張性も考慮して開発を進めます。
