# Tech Stack: AtCoder GUI

## 1. Core Runtime & Language
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Runtime:** [Node.js](https://nodejs.org/) (Target: >=18.0.0)
- **Package Manager:** [pnpm](https://pnpm.io/)

## 2. Browser Automation
- **Automation Framework:** [Playwright](https://playwright.dev/)
- **Utility:** [Cheerio](https://cheerio.js.org/) (Fast HTML parsing for static content)

## 3. Testing & Code Quality
- **Test Runner:** [Vitest](https://vitest.dev/)
- **Linter:** [ESLint](https://eslint.org/)
- **Formatter:** [Prettier](https://prettier.io/)
- **Git Hooks:** [Husky](https://typicode.github.io/husky/), [lint-staged](https://github.com/lint-staged/lint-staged)

## 4. Configuration & Templating
- **Config Management:** [conf](https://github.com/sindresorhus/conf)
- **Data Format:** [JSON5](https://json5.org/)
- **Templating Engine:** [Nunjucks](https://mozilla.github.io/nunjucks/)

## 5. Deployment & CI/CD
- **CI/CD:** [GitHub Actions](https://github.com/features/actions)
