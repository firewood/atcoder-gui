Problem Statement

A sequence of length N, a = {a_1, a_2, a_3, ..., a_N} is given.

Answer the sum of the sequence.

Constraints

  • N is an integer between 1 and 10 \ 000 (inclusive).
  • a_i is an integer between 1 and 1 \ 000 \ 000 \ 000 (inclusive).

Input

The input is given from Standard Input in the following format:

T
\text{case}_1
\text{case}_2
\vdots
\text{case}_T

Each test case is given in the following format:

N
a_1 a_2 a_3 ... a_N

Output

Print T lines.

Each line should be the sum of the sequence.

Sample Input 1

1
3
3 1 4

Sample Output 1

8

Sample Input 2

2
4
1 2 3 4
2
1 2

Sample Output 2

10
3
